# Content Selection Guide

Guide for selecting and using content from the Content Browser in your workflows.

## Overview

The Content Browser is designed to help you discover and select content for use in:
- **Process Builders**: Generate trivia sets, collections, etc.
- **Ideation Plans**: Build structured content plans
- **Collections**: Create Wisdom, Greetings, Motivational, Stats content
- **Trivia**: Generate Multiple Choice, True/False, Who Am I questions

## Selection Workflow

### 1. Discover Content

**Start with Themes:**
- Browse by theme to see what content is available
- Check theme statistics to see content volume
- Identify themes with rich content

**Use Search:**
- Search for specific topics or titles
- Combine search with theme filters
- Discover related content

**Review Summaries:**
- Read source summaries to understand content
- Check tags for additional context
- Verify content matches your needs

### 2. Evaluate Content

**Check Status:**
- Ensure content is "complete" (fully processed)
- Avoid "pending" or "failed" content
- Verify content is ready for use

**Review Usage:**
- Check usage badges to see where content has been used
- Identify unused content for new opportunities
- Avoid overusing the same content

**Assess Quality:**
- Review summaries for relevance
- Check tags for topic alignment
- Verify theme/category accuracy

### 3. Select Content

**For Process Builders:**
- Select content that matches your generation goal
- Consider theme alignment with target content type
- Choose diverse content for variety

**For Collections:**
- Select content appropriate for collection type
- Ensure theme matches collection theme
- Consider content that hasn't been used yet

**For Trivia:**
- Select content with interesting facts
- Choose content suitable for question generation
- Consider difficulty level of source content

## Content Evaluation Criteria

### Relevance

**Theme Alignment:**
- Does the theme match your goal?
- Is the category appropriate?
- Are tags relevant to your needs?

**Content Quality:**
- Is the summary clear and informative?
- Does it contain useful information?
- Is it suitable for your use case?

### Usage Status

**Unused Content:**
- Best for new content generation
- Fresh opportunities
- No duplication concerns

**Used Content:**
- Check where it's been used
- Consider if reuse is appropriate
- May indicate high-quality content

### Status Indicators

**Complete:**
- ✅ Ready to use
- Fully processed
- All metadata available

**Pending:**
- ⏳ Still processing
- Wait for completion
- May not have all metadata

**Failed:**
- ❌ Processing error
- May have incomplete data
- Review before using

## Selection Strategies

### Strategy 1: Theme-First

1. Choose a theme that matches your goal
2. Browse all content in that theme
3. Select multiple sources from the theme
4. Use for theme-specific generation

**Best For:**
- Themed collections
- Topic-focused trivia sets
- Consistent content generation

### Strategy 2: Search-First

1. Search for specific topics or keywords
2. Review search results across themes
3. Select relevant content
4. Use for topic-focused generation

**Best For:**
- Specific topic research
- Finding related content
- Cross-theme content discovery

### Strategy 3: Usage-Based

1. Filter to show unused content
2. Browse available unused sources
3. Select fresh content
4. Use for new content generation

**Best For:**
- Avoiding duplication
- Finding new opportunities
- Maximizing content utilization

### Strategy 4: Quality-First

1. Review summaries and tags
2. Identify high-quality sources
3. Select best content
4. Use for premium content generation

**Best For:**
- High-quality collections
- Featured content
- Premium trivia sets

## Using Selected Content

### In Process Builders

**Workflow:**
1. Select content in Content Browser
2. Navigate to Process Builder
3. Content is automatically staged (future)
4. Configure generation parameters
5. Generate content

**Considerations:**
- Ensure content theme matches builder goal
- Verify content status is "complete"
- Check content hasn't been overused

### In Ideation Plans

**Workflow:**
1. Select content in Content Browser
2. Add to ideation plan
3. Build structured plan
4. Export plan for execution

**Considerations:**
- Select diverse content for variety
- Consider content relationships
- Plan for content combination

### In Collections

**Workflow:**
1. Identify collection type (Wisdom, Greeting, etc.)
2. Select appropriate content
3. Use content in collection creation
4. Verify content fits collection theme

**Considerations:**
- Match content theme to collection
- Ensure content format is suitable
- Check content usage to avoid duplication

## Best Practices

### Content Diversity

**Avoid Overuse:**
- Don't use the same content repeatedly
- Check usage badges before selecting
- Rotate content selections

**Theme Variety:**
- Mix themes for diverse content
- Don't limit to single theme
- Explore cross-theme opportunities

### Quality Assurance

**Verify Before Use:**
- Check content status
- Review summaries
- Verify theme accuracy

**Test Content:**
- Preview generated content
- Verify quality meets standards
- Adjust selection if needed

### Workflow Efficiency

**Batch Selection:**
- Select multiple sources at once (future)
- Group related content
- Plan selection strategy

**Document Selections:**
- Note why content was selected
- Track selection patterns
- Learn from successful selections

## Common Use Cases

### Use Case 1: Generate Trivia Set

**Selection:**
- Choose content with interesting facts
- Select diverse themes
- Ensure content is "complete"

**Process:**
1. Select 5-10 sources
2. Navigate to Build Trivia Set
3. Configure trivia parameters
4. Generate questions

### Use Case 2: Create Wisdom Collection

**Selection:**
- Choose inspirational content
- Select Leadership & Staff theme
- Look for motivational summaries

**Process:**
1. Select relevant sources
2. Navigate to Collections
3. Create Wisdom entry
4. Use selected content

### Use Case 3: Build Themed Collection

**Selection:**
- Filter by specific theme
- Select multiple sources from theme
- Ensure content variety

**Process:**
1. Filter by theme
2. Select 3-5 sources
3. Create collection
4. Use selected content

## Troubleshooting

### No Suitable Content Found

**Solutions:**
- Broaden theme filters
- Try different search terms
- Check other themes
- Review content status

### Content Already Used

**Solutions:**
- Look for unused content
- Consider if reuse is appropriate
- Find similar content
- Use different themes

### Content Not Ready

**Solutions:**
- Wait for processing to complete
- Check content status
- Select alternative content
- Review processing logs
