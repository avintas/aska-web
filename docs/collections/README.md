## Overview

The collections system provides content management for:
- **Wisdom** - Philosophical musings (Penalty Box Philosopher)
- **Greetings** - Hockey Universal Greetings (HUGs)
- **Stats** - Hockey statistics
- **Motivational** - Motivational quotes
- **Hero Collections** - Curated 7-item collections
